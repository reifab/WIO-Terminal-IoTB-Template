{\footnotesize\ttfamily void update\+Interface\+Status (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Interface Icons updaten. 

Diese Methode updatet die Icons oben rechts in der Ecke. Ist eine SD Karte vorhanden, werden Bilder von der SD Karte verwendet. Ist keine SD Karte vorhanden, zeichnet diese Methode farbige Kreise.

\begin{DoxyNote}{Zu beachten}
Die Icons oder Kreise werden nur neu gezeichnet, wenn sich der Wert verändert hat. Dadurch kann diese Methode oft aufgerufen werden, ohne dass die Icons flackern. 
\end{DoxyNote}
