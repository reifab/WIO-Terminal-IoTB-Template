Io\+TB Bibliothek für den WIO Terminal Display. ~\newline
Diese Bibliothek bildet ein Grundgerüst für eine einfachere Nutzung des Displays. Mit dem File page.\+c können merhere Seiten vordefiniert werden. Diese Bibliothek ist dafür zuständig, dass die Seiten gezeichnet, aktualisiert und verändert werden können. ~\newline
Zusätzlich werden gewisse Interface Zustände angezeigt, sofern diese Verwendet werden. zu den Interface gehören den Wi\+Fi, MQTT und SD Karte. ~\newline
