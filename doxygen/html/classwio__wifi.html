◆ getNetworksFound()

int getNetworksFound ( void  )

Anzahl der gefundenen Netzwerke auslesen.

Diese Methode gibt den aktuellen Scan Status zurück oder, bei abgeschlossenem Scan, die Anzahl der gefundenen Netzwerke zurück.

Rückgabe
int Anzahl gefundene Netzwerke oder Scan Status
  • -2: Scan wurde noch nicht getriggert/ausgelöst
  • -1: Scan ist noch nicht abgeschlossen
  • >0: Anzahl der gefundenen Netzwerke
