◆ line_typ_e

enum line_typ_e

Enum für die verschiedenen Zeilentypen.

Noch zu erledigen:
Umrechnung von Value zu Zeit beschreiben
Aufzählungswerte
TEXT 

Zeigt einen Text an.

NUMERIC 

Zeigt einen Zahlenwert an. Die Nachkommastellen kann mit settings_e eingestellt werden. Kann auch eine Masseinheit anzeigen. Siehe line_t

BAR 

Zeigt einen Balken an, wie ein Ladebalken oder Balkendiagramm.

TIME 

Zeigt eine Zeit an. Das Zeitformat kann mit settings_e eingestellt werden.

END_LINE_TYP 

muss das letzte Element sein. Nicht verwenden!!!

◆ settings_e

enum settings_e

Enum für die verschiednenen Zeilen Einstellungen.

Aufzählungswerte
DEFAULT 

Wird verwendet, wenn keine Einstellung möglich oder um Default zu verwenden.

DECIMAL_PLACES_0 

Zeigt 0 Nachkommastellen an --> für line_typ_e NUMERIC

DECIMAL_PLACES_1 

Zeigt 1 Nachkommastellen an --> für line_typ_e NUMERIC

DECIMAL_PLACES_2 

Zeigt 2 Nachkommastellen an --> für line_typ_e NUMERIC, DEFAULT

DECIMAL_PLACES_3 

Zeigt 3 Nachkommastellen an --> für line_typ_e NUMERIC

BAR_EMPTY 

Zeigt ein Balken an, ohne einen Wert --> für line_typ_e BAR, DEFAULT.

BAR_SHOW_VALUE 

Zeigt ein Balken an, mit dem Balkenwert auf dem Balken --> für line_typ_e BAR.

TIME_HH_MM 

Zeitformat hh:mm, DEFAULT --> für line_typ_e TIME.

TIME_HH_MM_SS 

Zeitformat hh:mm:ss --> für line_typ_e TIME.

END_SETTING 

muss das letzte Element sein. Nicht verwenden!!!
