Taster

Das WIO Terminal besitzt an der Oberseite drei Taster. An der Front befindet sich ein 5-Weg-Schalter (Hoch, Runter, Links, Rechts, Drücken).

Defines für die Taster

Die Defines für die Taster lauten wie folgt:

Taster Define
Button A WIO_KEY_A
Button B WIO_KEY_B
Button C WIO_KEY_C
Hoch WIO_5S_UP
Runter WIO_5S_DOWN
Links WIO_5S_LEFT
Rechts WIO_5S_RIGHT
Drücken WIO_5S_PRESS
